# Stasis Photo Timeline

Live photo timeline for [Stasis](https://stasis.hackclub.com) a High School Hardware Hackathon, Austin TX, May 15–18 2025.

## Local dev

```bash
cd scraper
npm install
PHOTOS_ALBUM_URL="https://photos.app.goo.gl/..." node scrape.js
```

Then open `index.html` with a local server:
```bash
npx serve .
```

## Notes
- `data/photos.json` is auto-generated by the scraper, don't hand-edit it
- GitHub Actions re-scrapes every 6 hours and commits any new photos automatically
- To force a refresh: Actions > Scrape Google Photos > Run workflow
