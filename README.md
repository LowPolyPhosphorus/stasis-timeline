# Stasis Photo Timeline

A photo timeline for [Stasis](https://stasis.hackclub.com) — a High School Hardware Hackathon in Austin, TX, May 15–18 2026. Built by [LowPolyPhosphorus](https://github.com/LowPolyPhosphorus) as part of organizing the event with [Hack Club](https://hackclub.com).

Photos are sourced from attendee cameras, uploaded to the Hack Club CDN, and displayed on an interactive zoomable timeline spanning the full event — from day view down to minute-by-minute.

## Stack

- Vanilla HTML/CSS/JS — no framework
- Canvas-based timeline renderer with zoom, pan, and lightbox
- Photos hosted on Hack Club CDN
- Deployed on GitHub Pages
